📊 KPI Analytics Dashboard – Finance
📌 Overview

This project presents an interactive Finance KPI Dashboard developed using Power BI to monitor and evaluate business performance.

The dashboard focuses on key financial indicators such as actual sales, target sales, variance, and trends.

It provides a clear view of whether business goals are being achieved and highlights areas that require improvement.

🎯 Key Features

📊 Comparison of Actual Sales vs Target Sales

📉 Variance and Variance % analysis

📅 Monthly performance trends visualization

👥 Salesperson-wise performance tracking

📈 Trend indicators (sparklines)

🔍 Interactive filters (Team-based slicers)

🧾 Smart Narrative for automated insights
